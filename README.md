# payment-transaction-dashboard
💰Payment Transaction Analysis Dashboard | Python + Power BI📊
Analyzed 11,589 ATM withdrawal transactions spanning 7 years (2011–2017) to uncover spending patterns across festivals, ATM locations, card types, and days of the week.

Tools Used: Python (Pandas, NumPy) · Power BI

What I did:

Cleaned and preprocessed raw transaction data using Python — handled missing values, standardized date formats, removed duplicates, and engineered new features (Holiday Sequences, Festival categories)
Built an interactive Power BI dashboard with 6 visualizations covering withdrawal trends by year, ATM location, card type, day of week, and festival period

Key Insights:

NH (Non-Holiday) period accounts for 5.7bn in withdrawals — far exceeding all festival periods combined
2014 was the peak withdrawal year at 1.2bn, followed by a steady decline through 2017
KK Nagar ATM is the busiest location with 0.40M withdrawals
XYZ Card withdrawals consistently outperform Other Card across all years
Sunday is the highest withdrawal day (0.31M), with activity declining toward Friday
